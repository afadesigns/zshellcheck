# Community Governance

## Maintainers

The current maintainers of `zshellcheck` are:

- @afadesigns (Lead Maintainer)

## Decision Making

Major decisions (e.g., new features, breaking changes) are discussed via GitHub Issues and Pull Requests. Consensus among maintainers is sought.

## Roles

- **Contributor:** Submits PRs, reports bugs.
- **Maintainer:** Reviews PRs, merges code, manages releases.
